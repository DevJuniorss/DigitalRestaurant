# DigitalRestauran
Projeto Integrado em Engenharia de Software II  
Universidade Federal do Ceará – Campus Quixadá  

## 📌 Sobre o Projeto

**RU Digital** é um sistema web desenvolvido com foco na modernização da relação entre estudantes/docentes e o Restaurante Universitário (RU). A aplicação oferece funcionalidades como agendamento de consultas nutricionais, cadastro de refeições e gerenciamento de dietas personalizadas.

## 🎯 Objetivos

- Facilitar o acompanhamento nutricional no ambiente de ensino.
- Tornar o cardápio do restaurante mais acessível e organizado.
- Oferecer recursos personalizados para pacientes e nutricionistas.

## ⚙️ Funcionalidades Principais

- 📅 Agendamento e gerenciamento de consultas entre pacientes e nutricionistas.
- 🍽️ Cadastro e exibição do cardápio semanal do restaurante (café, almoço, jantar).
- 🥗 Criação e visualização de dietas personalizadas por pacientes.
- 👤 Sistema de autenticação com controle de permissões para diferentes perfis (nutricionista, paciente, administrador RU).

## 👤 Perfis de Usuários

- **Nutricionista**: Acompanha pacientes e cria dietas.
- **Paciente**: Aluno(a) que agenda consultas e visualiza sua dieta.
- **Administrador**: Gerencia o cardápio semanal do restaurante universitário.

## ✅ Requisitos Funcionais

- **RF01**: Agendar consulta com nutricionista.
- **RF02**: Editar ou cancelar consultas.
- **RF03**: Cadastrar refeições semanais do RU.
- **RF04**: Consultar cardápio do RU.
- **RF05**: Cadastrar dietas personalizadas.
- **RF06**: Visualizar dieta individual.
- **RF07**: Gerenciamento de usuários com autenticação e permissões.

## 🚫 Requisitos Não Funcionais

- **Usabilidade**: Interface responsiva e intuitiva.
- **Segurança**: Senhas criptografadas e controle de acesso por perfil.
- **Desempenho**: Suporte a até 100 usuários simultâneos.
- **Disponibilidade**: Sistema disponível 24/7 (exceto manutenção).

## 📖 Regras de Negócio

- Apenas nutricionistas podem criar/editar dietas.
- Apenas administradores do RU podem editar o cardápio.
- Cada paciente pode ter apenas uma dieta ativa por vez.
- Consultas devem ser agendadas com no mínimo 24h de antecedência.

## 📐 Tecnologias e Arquitetura

- Aplicação web responsiva.
- Arquitetura cliente-servidor.
- Banco de dados relacional para persistência de dados.
